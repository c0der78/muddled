Account

Represents a user account. An account may have multiple game characters. 

struct Account { 
    identifier_t id; 
    const char *login; 
    const char *email; 
    const char *password; 
    AccountPlayer *players; 
    int timezone; 
    identifier_t autologinId; 
    Flag *flags; 
    Forum *forum; 
    Note *inProgress; 
    Connection *conn; 
    Character *playing; 
    AccountForum *forumData; 
};
Fields
login

account id used to login

email

email address for account

password

encrypted password for account

players

a list of players linked to the account

timezone

the timezone of the account user

autologinId

id of the player to autologin with

flags

flags that modify account behaviour

forum

the current forum the account is reading. will never be null.

inProgress

the current note in progress. Can be null.

forumData

data cached from all forums

conn

the network connection descriptor

playing

the current player being played
