Affect

Represents an affect that can be applied to things in the world 

struct Affect { 
    Affect *next; 
    identifier_t id; 
    int from; 
    int level; 
    int duration; 
    int perm_duration; 
    int modifier; 
    Flag *flags; 
    affect_callback *callback; 
};
Fields
type

a skill id if the affect comes from a skill

level

the level of the affect

duration

the current duration of the affect

perm_duration

the duration when the affect was first created

modifier

the modifier value of the affect

callback

the callback method when the affect is applied
